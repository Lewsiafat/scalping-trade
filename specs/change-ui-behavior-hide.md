- **日期**: 2026-03-03
- **描述**: 實作 UI 區塊的折疊功能，並將折疊狀態儲存於用戶端 (localStorage)。支援折疊的區塊包含「交易設定」、「K 線圖表」與「細部分析結果（多時間框架及以下）」。

## 實作細節
1. 修改 `app_v2.py` 內的 HTML 結構，替換適當的標題為可點擊折疊的標題（例如加上 toggle button 或是 cursor-pointer）。
2. 在前端 CSS 增加 `.hidden` 或折疊用的過渡動畫 class。
3. 在前端 JavaScript 中實作折疊邏輯：
   - 讀取 localStorage 內的設定，初始化各區塊的展開/折疊狀態。
   - 點擊標題時，切換對應區塊的顯示狀態，同步更新 localStorage。
4. 測試各區塊（交易設定、K線圖表、細部分析）是否能正常折疊與展開。
