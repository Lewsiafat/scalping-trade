# 建立新主題 (create-custom-theme)

- **分支:** `feat/create-custom-theme`
- **日期:** 2026-03-03

## 描述
使用 theme-factory 建立一個新的自訂主題。設計要求為「簡約、柔和」，同時刻意避開 Tailwind CSS 預設外觀的風格，旨在提供一種更具溫潤感與獨特性的視覺體驗。

## 任務清單
- [x] 透過 `theme-factory` 產出柔和簡約風 (Soft Minimalist) 的自訂主題文件。
- [x] 決定配色方案：具備柔和對比度、避開高飽和度的基礎色調（例如：燕麥色、莫蘭迪色系）。
- [x] 決定字體組合：選擇相較於系統預設 sans-serif 更有溫度的字型（例如：標題使用 Serif 或圓潤字體，內文使用易讀且細膩的 Sans-serif）。
- [x] 寫入主題文件 `.agent/skills/theme-factory/themes/soft-minimalist.md`。
- [x] 向使用者展示生成結果以供審核。
- [x] 確認後套用至專案（如有指定目標檔案）。
